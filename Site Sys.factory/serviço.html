Nossos serviços

A system factory é uma empresa focada em criação de sistemas personalizados para cada situação difetente, para que assim possamos atender nossos clientes da forma mais precisa, entregando um sistema que atende todas as necessidades. Além disso qualquer tipo de suporte ou ajuste em um sistema é prontamente atendido pela nossa equipe. Nossos clientes são sempre atualizados de como está o sistema para que possamos juntos criarmos as melhores soluções para os problemas que queremos solucionar.

SYS.Aesthetic

Sobre o sistema

O sistema Aesthetic foi desenvolvido para uma clinica de estética onde não havia nenhum tipo de controle eletronico dos serviços prestados. Dentro dele é possivel cadastrar novos clientes ou funcionarios, imprimir recibos e será futuramente implementado uma agenda para facilitar os funcionarios a ver quais são as a fazeres do dia.
